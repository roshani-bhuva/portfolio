# Portfolio Website

A personal portfolio built with Vite, React, Tailwind CSS, and Framer Motion to showcase skills, projects, and experience.

## Features

- **Responsive design** — Works across common screen sizes.
- **Smooth animations** — Framer Motion for interactive transitions.
- **Polished UI** — Dark theme with a clear visual hierarchy.
- **SEO-friendly** — Meta tags and structure aimed at discoverability.

## Tech stack

- **Frontend:** React
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion

## Local setup

1. Install dependencies:

   ```sh
   npm install
   ```

2. Start the development server:

   ```sh
   npm run dev
   ```

3. Open the URL shown in the terminal (Vite defaults to port 5173 unless in use).

## Production build

```sh
npm run build
```

Preview the production build locally:

```sh
npm run preview
```

## Deployment

You can deploy the contents of the `dist` folder after `npm run build` to any static host you prefer (for example Netlify, Vercel, or similar).
